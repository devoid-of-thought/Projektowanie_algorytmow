# Sortowania

Projekt wykonany na potrzeby przedmiotu "Projektowanie i Analiza Algorytmów" na Politechnice Wrocławskiej.

## Opis projektu

Należało wykonać 3 sortowania:

- heapsort
- introsort
- timsort
- quicksort

Zostały zaimplementowane w C++
